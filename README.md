# Weather CLI - Command Line Weather Application

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![GitHub Stars](https://img.shields.io/github/stars/r123singh/weather-cli.svg)](https://github.com/r123singh/weather-cli/stargazers)
[![GitHub Issues](https://img.shields.io/github/issues/r123singh/weather-cli.svg)](https://github.com/r123singh/weather-cli/issues)

A powerful command-line weather application with premium features, providing real-time weather data, forecasts, and interactive maps.

## 🌟 Features

### Free Features
- Basic weather information for any city
- Current temperature and conditions
- Basic favorites management
- Simple notifications
- Basic weather maps

### Premium Features
- Advanced weather maps with animations
- Historical weather data
- Multiple city comparison
- Data export (CSV/JSON)
- Custom alerts and notifications
- Air quality index
- UV index
- Detailed 5-day forecast
- Interactive radar maps

## 🚀 Quick Start

### Installation
```bash
# Using pip
pip install weather-cli

# From source
git clone https://github.com/r123singh/weather-cli.git
cd weather-cli
pip install -e .
```

### Basic Usage
```bash
weather-cli
```
